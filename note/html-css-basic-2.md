## AM10:00~ 이고잉님 강의

- 공부할 때는 효율적으로 공부하는 것도 중요하지만 기분을 잘 챙기는 것도 중요하다. 
좋은 기분을 가지고 만족감을 충분히 누리도록 하자.
- 이고잉님은 경험을 먼저 하고 이론을 설명하는 것을 좋아한다.

- 중급자는 ‘눈치껏’ 공부하라
    - 대부분의 선생님들은 입문 단계에 포진해 있다. 중급 단계로 가면 없다.
    - 가설을 세우고, 확인해보고, 맞으면 넘어가는 등의 방식으로.
    - 선형적 접근 방식의 학습 방법은 비효율적이다.
    - 이러한 트레이닝이 입문단계에서 갖춰져야 좋다.
- 정확한 용어를 구사하자.
- 공학은 결과 지향, 과학은 과정 지향, 그럼에도 더 좋은 방법을 추구해 보자.

### 팁

- h1 쓰고 Tab을 누르면 자동으로 `<h1></h1>`가 완성된다. 다른 태그도 마찬가지!
- 특정 문자열 선택 → 보기 → 명령 팔레트 → Wrap with Abbreviation → 태그 입력 하면 해당 태그가 씌워짐. [https://ssimplay.tistory.com/420[](https://opentutorials.org/course/3265)](https://ssimplay.tistory.com/420) (Ctrl + Shift + P)
- `lorem1000` 단어인듯 아닌듯한 라틴어가 1000개 만들어짐.
